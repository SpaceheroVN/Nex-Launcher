# Nex v1.1.0

### ✨ TÍNH NĂNG MỚI

- **Hiển thị dạng lưới với phân loại chi tiết:** Phần mềm giờ được nhóm tự động theo danh mục cụ thể (Trình duyệt, Tiện ích, Công cụ phát triển...) thay vì chỉ chia Ứng dụng/Game, giúp tìm kiếm và quản lý dễ hơn.

- **Kéo thả file installer:** Hỗ trợ kéo thả trực tiếp file `.exe`, `.msi` vào cửa sổ để thêm phần mềm vào danh sách nhanh chóng.

- **Tự động đề xuất danh sách phần mềm:** Khi chạy lần đầu, ứng dụng gợi ý tải về danh sách phần mềm phổ biến để người dùng mới bắt đầu ngay.

### 🚀 CẢI TIẾN

- **Hộp thoại tiến trình chi tiết:** Hiển thị trạng thái cụ thể cho từng ứng dụng (đang tải, đang cài, hoàn tất, lỗi) kèm thông báo qua System Tray.
