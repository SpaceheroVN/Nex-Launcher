# Nex v1.2.0

### ✨ TÍNH NĂNG MỚI

- **Tích hợp Winget (Windows Package Manager):** Tìm kiếm và cài đặt phần mềm trực tiếp từ kho chính thức của Microsoft mà không cần tìm link download thủ công.

- **Cài đặt đa luồng (Multi-threading):** Cài nhiều chương trình đồng thời, tăng tốc đáng kể quá trình cài đặt hàng loạt. Giao diện không còn bị đơ/"treo" trong quá trình cài.

- **Launcher trung tâm:** Khi mở ứng dụng, hiển thị giao diện Launcher cho phép chọn công cụ (Installer) và quản lý cài đặt chung (ngôn ngữ, theme) tại một nơi.

### 🚀 CẢI TIẾN

- **Hộp thoại tiến trình được thiết kế lại hoàn toàn:** Hiển thị trạng thái chi tiết cho từng ứng dụng (đang tải, đang cài đặt, hoàn tất, lỗi) giúp theo dõi chính xác tiến trình.
