# Nex v1.3.0

### ✨ TÍNH NĂNG MỚI

- **Trình gỡ cài đặt (Uninstaller) hoàn toàn mới:**
  - Tự động quét và liệt kê tất cả phần mềm đã cài trên hệ thống.
  - Hiển thị icon ứng dụng, thông tin chi tiết (nhà phát hành, ngày cài đặt, dung lượng) và có bảng cài đặt riêng.
  - Hỗ trợ gỡ cài đặt im lặng (silent) hàng loạt.

### 🚀 CẢI TIẾN

- **Installer tự động retry:** Tự động thử lại khi tải hoặc cài đặt thất bại, giúp cài đặt hàng loạt ổn định hơn.
- **Kéo thả nâng cao:** Kéo thả file `.exe`, `.msi` vào cửa sổ để thêm phần mềm vào danh sách nhanh chóng.
- **Chọn nhiều mục (Uninstaller):** Hỗ trợ `Shift+Click` (chọn theo khoảng) và `Ctrl+Click` (chọn từng mục) để thao tác nhanh hơn.
- **Giao diện Launcher thiết kế lại:** Giao diện chính được làm mới hoàn toàn với phong cách hiện đại, tùy chỉnh theme.
